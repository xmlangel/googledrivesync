## [Korean]

### 릴리즈 노트 v1.0.2

Google Drive Sync v1.0.2 버전이 릴리즈되었습니다. 이번 업데이트에서는 Google 로그인 안정성 강화와 내비게이션 흐름 개선에 집중했습니다.

## 주요 개선 사항

### 1. Google 로그인 흐름 및 안정성 강화

- Google 로그인 과정에서 발생할 수 있는 다양한 오류 상황에 대한 처리를 강화했습니다.
- 로그 기록을 최적화하여 문제 발생 시 원인 파악이 용이하도록 개선되었습니다.
- 보안 저장소(EncryptedSharedPreferences) 접근 실패 시 원활한 동작을 위한 폴백(Fallback) 메커니즘을 적용했습니다.

### 2. 내비게이션 및 사용자 경험(UX) 최적화

- 대시보드와 계정 관리 화면 간의 내비게이션 백스택(Backstack) 구조를 최적화하여 앱 사용 흐름을 더욱 매끄럽게 개선했습니다.
- 계정 설정 후 자동으로 폴더 선택 화면으로 이동하는 로직의 안정성을 높였습니다.
- 활성 계정이 없을 때 대시보드에서 로딩 상태를 명확하게 표시하도록 개선하여 사용자 혼란을 줄였습니다.

## 수정 사항

- 특정 상황에서 앱 라이프사이클과 연동된 로그인 상태 체크 로직의 버그를 수정했습니다.
- 내비게이션 중복 호출을 방지하는 플래그를 도입하여 화면 전환의 안정성을 강화했습니다.
