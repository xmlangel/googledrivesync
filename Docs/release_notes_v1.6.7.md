# Google Drive Sync v1.6.7 릴리즈 노트

이번 버전은 로컬 폴더 생성 기능 추가 및 Android 10 환경에서의 저장소 접근 호환성 개선에 집중했습니다.

## 주요 변경 사항

### 1. 로컬 폴더 생성 기능 추가
- 앱 내에서 직접 로컬 저장공간에 새로운 폴더를 생성할 수 있는 기능을 추가했습니다.
- 이를 통해 동기화 대상을 더 유연하게 구성할 수 있습니다.

### 2. Android 10 (API 29) 호환성 개선
- Android 10의 레거시 저장소 접근(Legacy External Storage)을 지원하여 Scoped Storage 관련 호환성 문제를 해결했습니다.

### 3. 동기화 제외 로직 안정화
- 동기화 제외(Exclusion) 필터링 로직을 정교화하고 관련 테스트 케이스를 보강했습니다.
- Git 관련 메타데이터 경로(`.git/`, `.gitignore`, `.gitattributes`, `.gitmodules`, `.gitkeep`)를 기본 제외 대상으로 추가했습니다.
- 위 Git 경로는 로컬 스캔뿐 아니라 서버(Drive) 변경 감지/처리에서도 동기화 대상에서 제외됩니다.

### 4. 버전 상향
- 앱 버전을 `1.6.7`로 상향(`versionCode 32`)했습니다.
