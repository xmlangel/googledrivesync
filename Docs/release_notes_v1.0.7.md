# 릴리즈 노트 v1.0.7

v1.0.7 업데이트에서는 동기화 조건의 유연성을 높이고 문제 발생 시 원인 파악을 위한 로깅을 강화했습니다.

## 주요 변경 사항

### 동기화 로직 최적화 (정밀도 개선)

- **1초 임계값(Threshold) 도입:** 파일 시스템(Android FAT32/exFAT)과 서버 간의 미세한 타임스탬프 오차(밀리초 단위)로 인해 불필요한 동기화가 발생하는 현상을 방지하기 위해 1초 단위 비교 방식을 적용했습니다.
- **불필요한 쓰기 방지:** 변경 사항이 없는 파일에 대해 중복된 메타데이터 업데이트가 발생하지 않도록 로직을 보강했습니다.

### 디버깅 강화

- **상세 동기화 로그:** 어떤 파일이 왜 동기화 대상으로 감지되었는지(로컬/드라이브의 실제 타임스탬프 비교값)를 로그에 상세히 기록하도록 개선했습니다. 이를 통해 중복 동기화 문제의 원인을 더 정확히 추적할 수 있습니다.

## 버그 수정

- `SyncManager.kt` 내 임시 항목 생성 시 드라이브 수정 시간 초기화 로직 수정.
