# 릴리즈 노트 v1.3.0

v1.3.0 버전은 동기화 중 발생하는 치명적인 네트워크 오류 처리를 대폭 개선하여 전문성을 강화한 마이너 업데이트 버전입니다.

## 주요 변경 사항

### 오류 처리 및 복원력 강화

- **네트워크 오류 즉시 중단:** DNS 오류, 타임아웃 등 치명적인 네트워크 문제가 발생할 경우 동기화를 계속 시도하여 상태를 악화시키는 대신, 즉시 중단하고 안전하게 현재 상태를 보존합니다.
- **예외 처리 구조 최적화:** Google Drive API 호출 시 발생하는 예외를 상위 로직으로 투명하게 전달하여 오류의 정확한 원인을 파악하고 대응할 수 있도록 구조를 개선했습니다.
- **상세 한글 오류 메시지:** 네트워크 연결 끊김, DNS 이슈 등 각 상황에 맞는 구체적인 한글 오류 메시지를 제공하여 사용자가 문제를 쉽게 이해할 수 있도록 했습니다.

### 동기화 엔진 안정화

- **재귀적 스캔 로직 개선:** 폴더를 재귀적으로 스캔하는 과정에서 업로드 대기 항목 수집의 정확도를 높여 동기화 결과의 일관성을 확보했습니다.
- **상태 관리 최적화:** 동기화 진행 상황을 추적하는 내부 데이터 흐름을 정비하여 비정상 종료 시에도 데이터 무결성을 유지할 수 있도록 보강했습니다.

### 품질 보증 및 환경 개선

- **단위 테스트 강화:** 네트워크 장애 시나리오를 포함한 15개의 핵심 단위 테스트를 추가 및 업데이트하여 동기화 로직의 안정성을 검증했습니다.
- **테스트 환경 호환성 확보:** 최신 Android SDK 환경에서 Robolectric 테스트가 안정적으로 수행될 수 있도록 테스트 구성을 최적화했습니다.

## 기술적 개선

- **오류 식별 로직:** `isFatalNetworkError` 유틸리티를 도입하여 일시적인 오류와 중단이 필요한 치명적 오류를 명확히 구분합니다.
- **통합 테스트 통과:** 전체 45개의 테스트 스위트가 모두 정상 통과됨을 확인하여 기존 기능과의 호환성을 보장합니다.
