# 릴리즈 노트 v1.4.0

Google Drive Sync v1.4.0 버전이 릴리즈되었습니다. 이번 업데이트에서는 대규모 파일 환경에서의 동기화 성능을 획기적으로 개선하고 데이터 무결성을 강화하는 고급 동기화 기능을 도입했습니다.

## 주요 개선 사항

### 1. 차분 동기화 (Changes API) 도입
- Google Drive의 Changes API를 통합하여 마지막 동기화 이후의 서버 변경 사항만 추적합니다.
- 수천 개의 파일이 있는 대규모 폴더에서도 수 밀리초 내에 동기화 스캔을 완료하여 시스템 자원 소모를 최소화합니다.
- 증분 업데이트 기능을 통해 네트워크 트래픽을 효율적으로 관리합니다.

### 2. MD5 기반 콘텐츠 검증 및 최적화
- 파일 크기와 수정 시간뿐만 아니라 MD5 체크섬을 비교하여 실제 내용의 변경 여부를 판단합니다.
- 타임존 차이나 미세한 시간 편차로 인한 불필요한 업로드 및 다운로드를 방지하여 전송 효율을 극대화했습니다.
- 데이터 무결성 검증을 통해 동기화의 신뢰도를 높였습니다.

### 3. 자동 업로드 기능 강화
- 로컬에서 새 파일이 생성되거나 기존 파일이 수정될 때 즉시 Google Drive로 동기화하는 자동 업로드 옵션을 추가했습니다.
- 사용자 설정 화면에서 자동 업로드 활성화 여부를 선택할 수 있습니다.

## 수정 및 개선 사항
- 서버 측에서 파일이 이동되거나 삭제되었을 때 로컬 파일 저장소와 데이터베이스에 즉시 반영되도록 로직을 개선했습니다.
- 로컬에서 실수로 삭제된 파일을 서버로부터 자동으로 복구하는 안정성을 강화했습니다.
- 동기화 기록(History) 관리 방식을 최적화하여 과거 작업 내용을 보다 정확하게 기록합니다.
