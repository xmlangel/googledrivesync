# 릴리즈 노트 v1.0.4

Google Drive Sync v1.0.4 버전이 릴리즈되었습니다! 이번 업데이트에서는 특정 파일 형식 및 크기에 따른 동기화 에러를 해결하고 안정성을 더욱 높였습니다.

## 주요 개선 사항

### 1. 동기화 안정성 강화 및 버그 수정

- **0바이트 파일 처리 개선 (416 에러 수정)**: 크기가 0바이트인 빈 파일을 동기화할 때 발생하던 "416 Requested range not satisfiable" 에러를 해결했습니다. 이제 빈 파일도 에러 없이 정상적으로 동기화됩니다.
- **Google 도구 파일 지원 (403 에러 수정)**: Google Docs, Sheets, Slides 등 직접 다운로드가 제한된 파일들을 오피스 형식(Docx, Xlsx, Pptx) 또는 PDF로 자동 변환하여 다운로드하는 기능을 추가했습니다. 이를 통해 "403 Forbidden" 에러를 해결하고 모든 유형의 파일에 대한 동기화가 가능해졌습니다.

### 2. 테스트 및 품질 관리

- **Drive API 헬퍼 테스트 추가**: 구글 드라이브 API 연동을 담당하는 핵심 모듈에 대해 유닛 테스트를 추가하여, 향후 코드 변경 시에도 동기화 로직의 정확성을 보장할 수 있도록 했습니다.
