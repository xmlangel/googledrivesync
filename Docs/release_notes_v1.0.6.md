# 릴리즈 노트 v1.0.6

Google Drive Sync v1.0.6 버전이 릴리즈되었습니다. 이번 업데이트에서는 동기화 효율성 개선과 빌드 파이프라인의 안정성 확보에 집중했습니다.

## 주요 변경 사항

### 동기화 엔진 개선 (중복 동기화 해결)

- **변경 감지 로직 최적화:** 파일별 개별 타임스탬프 비교 방식을 도입하여, 실제 변경이 없는 파일에 대해 불필요한 동기화(업로드/다운로드)가 발생하던 문제를 해결했습니다.
- **메타데이터 정확도 향상:** Google Drive API에서 반환하는 서버 측 수정 시간을 정확히 기록하여 기기 간 시간 차이로 인한 오동작을 방지합니다.

### CI/CD (GitHub Action) 안정화

- **빌드 파이프라인 개선:** 테스트 결과 업로드 단계의 오류로 인해 작업 전체가 중단되던 문제를 수정하여 안정적인 릴리즈 프로세스를 확보했습니다.
- **릴리즈 노트 자동 생성 최적화:** 태그 기반 릴리즈 노트 병합 로직을 개선하여 릴리즈 시 안내 문구가 누락되지 않도록 보완했습니다.

## 기술적 개선

- `SyncManager.kt` 내의 중복 동기화 방지용 단위 테스트 추가.
- GitHub Action 워크플로우의 버전 추출 로직을 정적 방식에서 동적 방식으로 변경.
