# 릴리즈 노트 v1.1.0

v1.1.0 버전은 Obsidian 앱 연동 기능 추가와 UI 편의성이 개선된 마이너 업데이트 버전입니다.

## 주요 변경 사항

### Obsidian 앱 연동

- **신규 실행 아이콘:** 대시보드 우측 상단에 Obsidian 앱을 즉시 실행할 수 있는 아이콘을 추가했습니다.
- **앱 감지 로직 개선:** Android 11 이상의 패키지 가시성 제한 문제를 해결하여 설치된 Obsidian 앱을 정확하게 감지합니다.
- **맞춤형 아이콘 적용:** 앱 디자인과 조화로운 투명 배경의 Obsidian 전용 아이콘을 적용했습니다.

### UI 및 사용성 개선

- **동적 버전 표시:** 앱 상단 바의 버전 정보를 하드코딩 방식에서 시스템 정보를 직접 가져오는 방식으로 변경하여 정확성을 높였습니다.
- **아이콘 간격 최적화:** 상단 바 액션 아이콘들의 간격을 조정하여 보다 조밀하고 정돈된 레이아웃을 제공합니다.

### 빌드 및 버전 관리

- **버전 업데이트:** 마이너 버전을 v1.1.0으로 업데이트하고 빌드 번호를 갱신했습니다.

## 기술적 개선

- `AndroidManifest.xml`에 `queries` 요소를 추가하여 외부 앱 실행 권한을 관리합니다.
- Compose의 `LocalContext`와 `PackageManager`를 활용하여 앱 메타데이터를 동적으로 렌더링합니다.
