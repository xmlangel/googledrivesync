# 릴리즈 노트 v1.2.0

v1.2.0 버전은 동기화 엔진의 정확도를 높이기 위한 MD5 체크섬 도입과 파일 이동/이름 변경 감지 기능이 추가된 마이너 업데이트 버전입니다.

## 주요 변경 사항

### 동기화 엔진 고도화

- **MD5 체크섬 기반 동기화:** 파일의 크기뿐만 아니라 MD5 체크섬을 비교하여 파일의 동일성을 완벽하게 보장합니다.
- **파일 이동 및 이름 변경 감지:** Google Drive 또는 로컬에서 파일이 이동하거나 이름이 변경된 경우, 이를 감지하여 불필요한 재업로드/다운로드 없이 동기화를 수행합니다.
- **용량 기반 최적화:** 대용량 파일에 대한 동기화 분석 속도를 개선하고 불필요한 데이터 전송을 최소화했습니다.

### 사용자 편의성 및 안전성 개선

- **업로드 확인 다이얼로그:** 파일이 업로드되기 전 사용자에게 확인을 요청하는 다이얼로그를 추가하여 의도치 않은 데이터 변경을 방지합니다.
- **기본 동기화 방향 조정:** 데이터 유실 방지를 위해 기본 동기화 방향을 '다운로드 전용'으로 설정했습니다.
- **상세 동기화 상태 표시:** 동기화 진행 중 어떤 작업(업로드, 다운로드, 이동 등)이 수행되고 있는지 상세 메시지를 통해 확인할 수 있습니다.

### 도구 및 자동화

- **배포 자동화 스크립트:** 빌드된 APK를 기기에 즉시 설치하고 버전을 검증할 수 있는 `deploy_apk.sh` 스크립트를 추가했습니다.
- **버전 관리 자동화:** `number_up.sh` 스크립트를 통해 메이저/마이너/빌드 버전을 손쉽게 관리할 수 있습니다.

## 기술적 개선

- **데이터베이스 스키마 업데이트:** MD5 체크섬 저장을 위한 `SyncItemEntity` 컬럼이 추가되었습니다.
- **로그 정규화:** 에러 확인을 용이하게 하기 위해 에러 로그 앞에 `[ERROR]` 접두사를 추가했습니다.
