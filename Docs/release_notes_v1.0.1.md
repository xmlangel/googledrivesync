# 릴리즈 노트 v1.0.1

Google Drive Sync v1.0.1 버전이 릴리즈되었습니다! 이번 업데이트에서는 대규모 파일 처리 및 디렉토리 구조 동기화 기능이 크게 개선되었습니다.

## 🚀 주요 개선 사항

### 1. 재귀적 디렉토리 동기화 지원

- 이제 선택한 폴더뿐만 아니라 하위 폴더의 모든 내용까지 재귀적으로 탐색하여 동기화합니다.
- 복잡한 폴더 구조를 그대로 유지하며 Google Drive와 동기화할 수 있습니다.

### 2. 구글 드라이브 파일 목록 조회 성능 개선 (Pagination 적용)

- Google Drive API를 통한 파일 목록 조회 시 페이지네이션(Pagination)을 적용했습니다.
- 수천 개 이상의 대량 파일이 포함된 폴더도 누락 없이 안정적으로 동기화할 수 있도록 개선되었습니다.

## 🛠 수정 사항

- 비정상적으로 종료될 수 있는 일부 파일 조회 로직의 안정성을 강화하였습니다.
