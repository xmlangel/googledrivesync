# 릴리즈 노트 v1.0.5

Google Drive Sync v1.0.5 버전이 릴리즈되었습니다. 이번 업데이트에서는 알림 소음 문제 해결과 동기화 작업의 안정성 강화에 집중했습니다.

## 주요 개선 사항

### 1. 알림 시스템 최적화 및 안정성 강화

- 동기화 시작 및 진행 과정에서 발생하는 "Muting recently noisy" 시스템 오류를 해결했습니다.
- 알림 업데이트 빈도를 최적화하여 안드로이드 시스템에 의해 알림이 차단되는 현상을 방지했습니다.
- 백그라운드 작업 및 수동 동기화 요청 시 알림 ID 관리 로직을 개선하여 사용자에게 더 정확한 상태 정보를 제공합니다.

### 2. 동기화 작업 정책 개선 (WorkManager)

- 중복 동기화 방지(Unique Work) 로직을 적용하여 여러 동기화 작업이 동시에 실행되는 문제를 해결했습니다.
- 앱 시작 시 주기적 동기화 작업이 불필요하게 재등록되는 로직을 수정하여 시스템 자원 소모를 줄이고 안정성을 높였습니다.

## 수정 사항

- 동기화 완료 알림이 짧은 간격으로 여러 번 갱신되는 버그를 수정했습니다.
- 알림 전송 빈도에 대한 자동화 검증 테스트를 도입하여 향후 재발 방지를 위한 기반을 마련했습니다.
