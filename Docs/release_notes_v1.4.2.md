# 릴리즈 노트 v1.4.2

Google Drive Sync v1.4.2 버전이 릴리즈되었습니다. 이번 업데이트에서는 사용자 인터페이스 개선, 코드 품질 및 테스트 커버리지 강화, 그리고 네트워크 오류에 대한 안정성을 개선했습니다.

## 주요 개선 사항

### 1. 사용자 인터페이스(UI) 및 편의성 개선
- 앱 종료 시 즉시 종료되지 않고 백그라운드로 전환되도록 동작을 변경했습니다.
- 앱 우측 상단에 명시적인 '종료' 버튼을 추가했습니다.
- 종료 버튼 클릭 시 확인 다이얼로그를 통해 실수로 앱을 닫는 것을 방지했습니다.

### 2. 코드 품질 및 테스트 강화
- JaCoCo를 도입하여 단위 테스트 커버리지 측정 환경을 구축했습니다.
- `AppVersionUtil`, `FileUtils`, `SyncLogger` 등 유틸리티 클래스에 대한 테스트 커버리지를 100% 달성했습니다.
- `GoogleAccount` 데이터 모델 및 관련 컴포넌트에 대한 단위 테스트를 추가했습니다.

### 3. 동기화 안정성 개선
- DNS 확인 실패 및 네트워크 관련 오류 발생 시 전체 동기화가 중단되지 않고 개별 오류를 로깅하며 계속 진행되도록 복원력을 강화했습니다.
- 네트워크 연결 오류 처리를 보다 정교하게 개선하여 동기화 작업의 신뢰도를 높였습니다.

## 기술적 변경 사항
- 프로젝트 전반에 걸쳐 코드 품질 향상을 위한 리팩토링 및 테스트 코드 최적화를 수행했습니다.
- JaCoCo 리포트 생성 가이드 문서를 추가하여 개발 편의성을 높였습니다.
