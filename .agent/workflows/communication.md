# 커뮤니케이션 지침

에이전트는 사용자와 소통할 때 다음 규칙을 엄격히 준수해야 합니다.

## 1. 기본 언어 설정

- **모든 답변은 한국어(Korean)로 작성합니다.**
- 기술적인 용어나 코드 설명 중 필요한 경우 영문을 혼용할 수 있으나, 주요 설명과 문장은 반드시 한국어로 작성합니다.

## 2. 응답 스타일

- 명확하고 간결하며 도움이 되는 톤을 유지합니다.
- 복잡한 설명은 불렛 포인트나 번호 매기기를 활용하여 가독성을 높입니다.
- 사용자가 요청한 작업의 맥락을 충분히 이해했음을 보여주는 응답을 제공합니다.
