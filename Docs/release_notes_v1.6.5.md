# Google Drive Sync v1.6.5 릴리즈 노트

이번 버전은 debug 단위 테스트 안정화와 동기화 품질 개선에 중점을 두었습니다.

## 주요 변경 사항

### 1. 동기화 중 폴더 삭제 시 즉시 중단
- 동기화 진행 중 대상 동기화 폴더가 삭제되면 작업을 즉시 중단하도록 개선했습니다.
- 사용자에게 동기화 중단 사유를 메시지로 안내합니다.

### 2. 로컬 동기화 루트 유실 감지 강화
- 동기화 중 로컬 루트 폴더가 삭제/유실된 경우 동기화를 계속 진행하지 않고 즉시 중단하도록 보강했습니다.

### 3. 기본 동기화 정책 개선 (권장: 다운로드 전용)
- 신규 동기화 폴더의 기본 정책을 다운로드 전용 중심으로 정비했습니다.
- 동기화 이력이 없는 초기 상태에서는 업로드/삭제로 인한 오동작을 방지하도록 보호 로직을 강화했습니다.

### 4. 초기 연결 시 중복/삭제 위험 경고 강화
- 기존 Drive 폴더 연결 시 로컬 데이터 삭제 여부를 사용자에게 명확히 선택받도록 개선했습니다.
- 삭제하지 않고 진행할 경우 중복 업로드/삭제 가능성 경고를 함께 표시합니다.

### 5. 폴더 선택 화면 기능 확장
- Drive 폴더 선택 화면에서 새 폴더를 바로 생성할 수 있는 기능을 추가했습니다.
- 기존 폴더 연결 시 안내/선택 UX를 개선해 안전한 초기 동기화를 유도합니다.

## 안정성 및 품질 개선
- 동기화 이동/이름 변경 처리 안정성 보강
- 동기화 관련 로그/가이드 정비
