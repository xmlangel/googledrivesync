# 릴리즈 노트 v1.0.0 (최초 배포)

**Google Drive Sync**의 첫 번째 공식 릴리즈를 발표하게 되어 기쁩니다! 이 버전은 안드로이드 기기와 Google Drive 간의 원활한 데이터 동기화를 위한 핵심 기능을 모두 포함하고 있습니다.

## 🚀 주요 기능

### 1. 양방향 동기화 (Bidirectional Sync)

- 로컬 폴더와 Google Drive 폴더 간의 변경 사항을 실시간으로 감지하고 동기화합니다.
- 파일의 수정 시간(Timestamp)을 비교하여 최신 버전의 데이터를 유지합니다.

### 2. 구글 계정 다중 지원

- 여러 개의 Google 계정을 앱에 연결하고 관리할 수 있습니다.
- 각 동기화 쌍(Folder Pair)마다 서로 다른 계정을 지정하여 사용할 수 있습니다.

### 3. 유연한 충돌 해결 정책

- 로컬과 서버 모두 변경된 경우, 사용자가 직접 선택할 수 있는 3가지 옵션을 제공합니다:
  - **로컬 우선 (Use Local)**: 로컬 파일로 드라이브를 업데이트합니다.
  - **드라이브 우선 (Use Drive)**: 드라이브 파일로 로컬을 업데이트합니다.
  - **모두 유지 (Keep Both)**: 로컬 파일 이름을 변경하여 두 버전을 모두 보존합니다.

### 4. 자동 백그라운드 동기화

- WorkManager를 사용하여 앱이 실행 중이지 않을 때도 백그라운드에서 주기적으로 동기화를 수행합니다.
- **실행 제약 조건 설정**: Wi-Fi 연결 시에만, 혹은 충전 중에만 동기화되도록 설정하여 데이터와 배터리를 절약할 수 있습니다.

### 5. 동기화 로그 및 내역 확인

- 모든 동기화 작업의 성공, 실패, 건너뜀 내역을 상세 로그를 통해 확인할 수 있습니다.

## 🛠 아키텍처 및 내부 개선

- **Jetpack Compose**: 최신 Android UI 프레임워크를 적용하여 매끄럽고 현대적인 디자인을 제공합니다.
- **Room Persistence**: 로컬 설정 및 동기화 상태를 안전하고 효율적으로 저장합니다.
- **CI/CD 통합**: GitHub Actions를 통한 자동 빌드 시스템을 구축하여 개발 효율성을 높였습니다.

## 📦 설치 안내

- GitHub Actions를 통해 빌드된 `app-debug.apk`를 아티팩트 혹은 릴리즈 페이지에서 다운로드하여 설치하실 수 있습니다.
