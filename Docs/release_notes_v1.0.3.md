# 릴리즈 노트 v1.0.3

Google Drive Sync v1.0.3 버전이 릴리즈되었습니다! 이번 업데이트에서는 동기화 엔진의 안정성을 강화하고, 상세한 진행 상황 확인 및 로그 관리 기능을 추가했습니다.

## 주요 개선 사항

### 1. 동기화 핵심 엔진 및 안정성 고도화

- 이미 동기화된 파일 및 폴더에 대한 처리 로직을 개선하여 중복 작업을 방지하고 효율성을 높였습니다.
- 다운로드되는 파일명의 특수문자 등을 정규화(Sanitize)하여 로컬 시스템과의 호환성을 강화했습니다.
- 파일 비교 로직을 별도 모듈로 분리하여 보다 정밀하고 빠른 동기화 여부 판단이 가능해졌습니다.
- 동기화 관리자(SyncManager)를 싱클톤 구조로 개편하여 앱 전체에서 일관된 동기화 상태를 유지하도록 개선했습니다.

### 2. 정밀한 진행 상태 및 로그 관리

- 동기화 시작 시 전체 파일 개수를 미리 파악하여 보다 정확한 백분율(%) 기반의 진행 상황을 표시합니다.
- 동기화 과정 중 발생하는 상세한 로그를 UI에서 바로 확인하고 외부로 공유할 수 있는 기능을 추가했습니다.
- 로그 화면에서 오류 발생 항목만 별도로 모아볼 수 있는 필터링 기능을 도입했습니다.

### 3. 미디어 및 문서 지원 확대

- 고도화된 MIME 타입 유틸리티를 통해 다양한 이미지, 오디오, 비디오 및 문서 형식에 대한 지원을 대폭 확장했습니다.

## 기술적 개선 및 테스트

- 프로젝트 전반에 걸친 단위 테스트 환경을 구축하고 주요 로컬 저장소 및 유틸리티에 대한 테스트 코드를 작성하여 코드 품질을 강화했습니다.
- 에이전트 기반의 개발 및 릴리즈 프로세스를 자동화하여 향후 빠른 기능 대응이 가능하도록 인프라를 개선했습니다.
