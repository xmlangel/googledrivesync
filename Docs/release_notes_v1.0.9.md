# 릴리즈 노트 v1.0.9

v1.0.9 버전은 불필요한 동기화가 발생하는 마지막 케이스까지 잡아낸 최종 최적화 버전입니다.

## 주요 변경 사항

### 크기 기반 동기화 최적화 (Final Fix)

- **신규 항목 크기 매칭:** 데이터베이스에 정보가 없는 초기 상태에서도 로컬과 드라이브의 파일 크기가 완벽히 일치하면 동일 파일로 간주해 즉시 연결합니다.
- **내용 보존형 업데이트 (Swallowing):** Obsidian 등 다른 앱에 의해 파일 내용은 그대로인데 수정 시간만 업데이트된 경우, 데이터를 다시 주고받지 않고 데이터베이스의 시간 값만 갱신합니다.
- **불필요한 트래픽 제로:** 파일 내용(Size)이 실제로 변하지 않았다면 어떠한 업로드/다운로드도 발생하지 않도록 개선되었습니다.

### UI 개선

- 상단 바의 버전 표기를 `v1.0.9`로 업데이트했습니다.

## 기술적 개선

- 로그 분석을 통해 파악된 '시간 기반 변경 감지의 한계'를 '크기 및 정황 기반 매칭'으로 극복했습니다.
